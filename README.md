# CodeAlpha Task 1 - Basic Calculator Program

## 📌 Task Description
This is Task 1 for the CodeAlpha C Programming Internship. 
The objective is to write a C program that performs basic arithmetic operations using `switch case`.

## ✨ Features
- **Operations supported**: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- **User input**: Takes operator and two numbers from user
- **Error handling**: 
    - Handles division by zero
    - Handles invalid operators
- **Formatted output**: Shows result up to 2 decimal places
